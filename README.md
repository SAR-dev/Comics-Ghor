# Project Title

Social Networking Website with Node.js as Backend and React.js as frontend

See the [live version!](https://comicsghor-3a6a7.firebaseapp.com/)

## Usage
* Backend
```python
edit .env file with required info
cd nodeapi/
npm install
npm run dev
```
* Frontend
```python
edit .env file with required info
cd reactfront/
npm install
npm start
```
